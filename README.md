# chatbot
一个可以进行训练的聊天机器人

#关于语料的说明

本次训练的语料是从互联网上找到的shooter的训练语料，语料质量很差劲，仅作为演示代码来用，大家可以使用自己的语料
语料下载地址：https://pan.baidu.com/s/1kWYIOVt,将文件下载后放到data目录下
